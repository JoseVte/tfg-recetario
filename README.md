# URIs

## User

#### Paginacion

```
GET /employees?page=2
```

#### Consultar uno

```
GET /employees/1
```

#### Crear uno

```
POST /employees
```

#### Actualizar datos

```
PUT/PATCH /employees
```

#### Borrar

```
DELETE /employees/1
```
